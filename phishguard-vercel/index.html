Setup Instructions
5 minutes to protection
1
Open Chrome and go to chrome://extensions
2
Enable Developer mode toggle (top-right corner)
3
Click Load unpacked and select your extension/ folder
4
Set your API URL below and regenerate the files
5
Open Gmail — PhishGuard now scans every email you open ✅
// manifest.json — paste into extension folder
{
  "manifest_version": 3,
  "name": "PhishGuard AI for Gmail",
  "host_permissions": ["https://mail.google.com/*", "http://Vercel API/*"],
  "content_scripts": [{"matches": ["https://mail.google.com/*"], "js": ["content.js"], "css": ["content.css"]}],
  "background": {"service_worker": "background.js"},
  "action": {"default_popup": "popup.html"}
}

Deployment Steps
Google Apps Script deployment
1
Go to script.google.com → Create new project
2
Enter your API URL above, click Generate, paste code into Code.gs
3
Also paste appsscript.json manifest (enable it in Project Settings)
4
Deploy → New deployment → Type: Gmail Add-on
5
For org-wide: Google Admin Console → Apps → Marketplace → Domain install
